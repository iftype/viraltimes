# Quiz feature

- 목적: 사용자가 밈을 이미 아는지와 ViralOrigin의 설명을 본 뒤 이해에 도움이 됐는지를 분리 측정한다.
- 한 세션은 최대 5개 published 사전 항목을 무작위 순서로 보여준다.
- 판단 응답은 `know`/`dont_know`, 정보 수요는 `view_detail`, 설명 효과는 `helpful`/`not_helpful`이다.
- 원시 이벤트는 보존하되 통계의 인지도는 같은 익명 세션·카드의 마지막 판단만, 상세 열람은 중복 없이 포함한다. 도움률 분모에는 마지막 설명 효과 응답만 포함한다.
- 브라우저 localStorage의 무작위 세션 ID만 사용하며 이름·이메일·계정 정보와 연결하지 않는다.
- 썸네일이 없으면 accent 기반 placeholder를 보여주고, 설명이 없으면 수집 중 문구를 사용한다.
- 퀴즈 카드와 누적 수치는 공개된 서버 데이터만 사용하며, API 실패 시 가짜 샘플 통계를 표시하지 않는다.
- 모바일 퀴즈는 헤더·푸터를 숨기고 `100dvh` 안에서 카드와 조작 버튼을 모두 보여 페이지 스크롤을 만들지 않는다.
